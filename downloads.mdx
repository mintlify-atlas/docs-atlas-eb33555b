### PPA (Ubuntu/Debian)

For stable packages on Ubuntu and Debian-based distributions:

<Tabs>
  <Tab title="Ubuntu">
    ```bash
    sudo add-apt-repository ppa:nicotine-team/stable
    sudo apt update
    sudo apt install nicotine
    ```
    
    Works on Ubuntu and distributions based on it (Linux Mint, elementary OS, Pop!_OS, Ubuntu flavors).
  </Tab>
  
  <Tab title="Debian">
    ```bash
    sudo apt update
    sudo apt install python3-launchpadlib software-properties-common
    sudo add-apt-repository 'deb https://ppa.launchpadcontent.net/nicotine-team/stable/ubuntu jammy main'
    sudo apt update
    sudo apt install nicotine
    ```
    
    Works on Debian and distributions based on it (Devuan, Peppermint OS).
  </Tab>
  
  <Tab title="Manual .deb">
    If you prefer to install a .deb package manually:
    
    1. [Download the .deb package](https://github.com/nicotine-plus/nicotine-plus/releases/latest/download/debian-package.zip)
    2. Install with `sudo dpkg -i nicotine_*.deb`
    
    <Warning>
      When installing manually, Nicotine+ will **not** update automatically. Download and install the latest version after every update.
    </Warning>
  </Tab>
</Tabs>

### PyPI (Universal)

If no package is available for your operating system, install from [PyPI](https://pypi.org/project/nicotine-plus/):

<Steps>
  <Step title="Install runtime dependencies">
    Ensure the [runtime dependencies](https://github.com/nicotine-plus/nicotine-plus/blob/master/doc/DEPENDENCIES.md) are installed for your system.
    
    For example, on Debian/Ubuntu:
    ```bash
    sudo apt install gir1.2-gtk-4.0 gir1.2-adw-1 python3-gi python3-gi-cairo
    ```
  </Step>
  
  <Step title="Install Nicotine+ from PyPI">
    ```bash
    pip3 install nicotine-plus
    ```
  </Step>
  
  <Step title="Update when needed">
    Nicotine+ will not update automatically. When a new release is available:
    ```bash
    pip3 install --upgrade nicotine-plus
    ```
  </Step>
</Steps>

## System Requirements

### Required Dependencies

- **Python 3.9 or newer** - Runtime language
- **GTK 4.6.9+** or **GTK 3.24.24+** - Graphical interface
- **PyGObject** - Python bindings for GTK

### Recommended Dependencies

- **libadwaita** - For Adwaita theme on GNOME (GTK 4)
- **gspell** - For spell checking in chat (GTK 3)

<Tip>
  Windows and macOS installers include all required dependencies - no additional setup needed!
</Tip>
